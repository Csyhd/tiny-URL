# **短網址產生器**
提供產生短網址服務(好像也沒變多短XDD)

## 功能
- 輸入網址送出可以產生短網址
- 可以按copy直接複製產生出來的短網址
- 可以直接點擊產生出來的短網址跳轉


## 如何使用

1.下載
    + 使用Terminal  輸入 git clone https://github.com/Csyhd/tiny-URL.git

2.進入
    + 使用cd 進入到下載完成的資料夾

3.安裝使用套件
    + npm install

4.執行
    + npm run dev 

5.執行成功
    + Terminal 顯示start

開始使用吧！
